# Block Jam 3D Clone

Block Jam 3D oyununun Unity ile geliştirilmiş klonudur. Bu proje sadece öğrenme ve deneme amaçlı yapılmıştır.

## Proje Hakkında

Bu proje, popüler mobil oyun Block Jam 3D'nin mekaniklerini anlamak ve Unity ile oyun geliştirme becerilerini geliştirmek için oluşturulmuştur.

## Özellikler

- Grid tabanlı karakter yerleşim sistemi
- A* pathfinding algoritması ile yol bulma
- Object pooling sistemi
- Event-driven mimari

## Teknik Detaylar

- **Unity Version:** 6000.0.59f2+
- **DOTween:** Animasyon sistemi
- **Grid System:** Custom grid yönetimi
- **Pooling:** Performans optimizasyonu

## Kullanım

Bu proje ticari amaçlı kullanıma uygun değildir. Sadece eğitim ve öğrenme amaçlıdır.
